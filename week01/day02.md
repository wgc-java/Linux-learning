## 一、核心概念

### 1. 用户与组的关系
- **一对一**：一个用户必须属于一个初始组（主组）
- **一对多**：一个用户可以加入多个附加组
- **多对一**：一个组可以有多个用户

### 2. 用户分类
| 类型 | UID | 说明 |
|------|-----|------|
| root | 0 | 超级管理员，拥有所有权限 |
| 系统用户 | 1-999 | 服务进程使用，如bin, daemon |
| 普通用户 | 1000+ | 我们创建的用户 |

## 二、常用命令速查

### 1. 用户创建与删除
```bash
# 创建用户（基本）
useradd wgc

# 创建用户并创建家目录（推荐）
useradd -m wgc

# 指定shell（防止无法登录）
useradd -m -s /bin/bash wgc

# 指定UID
useradd -u 1005 wgc

# 指定主组
useradd -g developers wgc

# 删除用户（保留家目录）
userdel wgc

# 删除用户及家目录（彻底删除）
userdel -r wgc


# 切换用户（加载环境变量）
su - wgc

# 切换用户（不加载环境变量，当前目录不变）
su wgc

# 切换到root
su -

# 临时获取root权限（推荐，需输入自己密码）
sudo ls /root

# 查看当前用户
whoami
id
