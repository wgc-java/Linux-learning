# 增加磁盘应用实例
首先输入指令 lsblk查看磁盘情况
发现现在有一个磁盘“sda”，这是创建该系统时分配的磁盘，
此时要再创建一个磁盘，
然后打开虚拟机vmware的设置，添加一个磁盘
添加完成后，开机，打开终端输入指令 fdisk /dev/sdb
开始对/sdb分区
* m显示命令列表，
* p显示磁盘分区 同fdisk -l
* n新增分区
* d删除分区
* w写入分区
* 说明：开始分区后输入n，新增分区然后选择p，分区类型为主分区。两次回车默认剩余
全部空间。最后输入w写入分区并退出，若不保存退出输入q。
开始分区后，选择分区号决定分几个区。然后w退出。
然后再用lsblk查看分区情况。

分区完成后，就要格式化该硬盘，赋予该硬盘id。

格式化硬盘：
mkfs -t ext4 /dev/sdb1/

查看后发现有id了

然后就可以开始挂载了
要在Linux上新建一个newdisk文件（要在根目录/上开始  cd/）
用mkdir newdisk
然后开始挂载
用  mount /dev/sdb1 /newdisk/
然后lsblk -f
发现挂载好了




如果想挂载到别的目录，他的指令叫卸载
umount 设备名称  挂载目录

# 注意 
用命令行挂载后重启后会失效


所以要永久挂载
通过修改/etc/fstab实现挂载
添加完成后 执行mount -a即刻生效
vim /etc/fstab
加挂载关系：/dev/sdb1        /newdisk     ext4    后面改成0 0 
意思分别为是否按照一定顺序进行文件系统检查和是否进行文件系统备份
保存退出然后reboot重启再查看发现已经挂载成功
